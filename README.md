# W Cleaning Web

Aplicación móvil desarrollada con Flutter para servicios de limpieza profesional.

## 📱 Requisitos
- Android 5.0 o superior
- 60 MB de espacio libre

## 🚀 Características
- Interfaz intuitiva
- Sistema de reservas
- Gestión de servicios
- Notificaciones

```bash
flutter pub get
```

## 🔧 Pasos para publicar la APP

Para los desarrolladores, necesitan:

1. Generar el APK:
```bash
flutter build apk --release
```

2. Los archivos importantes están en:
- `build/app/outputs/flutter-apk/app-release.apk`

3. Subir a GitHub:
- Crear nuevo release
- Subir el archivo APK generado
- Publicar el release

## Instrucciones para desarrolladores

Para clonar y ejecutar este proyecto:

```bash
git clone https://github.com/Leonard-3126/web-descargas.git
cd web-descargas
flutter pub get
flutter run
```
